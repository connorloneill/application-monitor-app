Respond with ONLY a valid JSON object in this exact format:
{
  "summary": "Concise explanation of the root cause and its impact",
  "rootCause": "Detailed technical explanation of why this issue occurs",
  "suggestedFix": "Description of the recommended code changes to resolve the issue",
  "codeSnippets": [
    {
      "filePath": "path/to/file.ts",
      "startLine": 42,
      "endLine": 55,
      "content": "the exact code from those lines",
      "explanation": "Why this specific code is causing the issue",
      "confidence": 0.85
    }
  ]
}

Guidelines:
- Identify 1-5 code snippets most responsible for the issue
- Include the exact code content from the files provided
- Use accurate line numbers from the source files
- Assign confidence scores: 0.9+ for certain, 0.7-0.9 for likely, below 0.7 for possible
- The suggested fix should be specific and actionable
- Consider race conditions, edge cases, null/undefined handling, type mismatches, and resource leaks
- If the issue spans multiple files, include snippets from each relevant file
